# Contourline-Strategic-BI
Strategic BI em Streamlit com IA aplicada a Vendas e Leads Perdidos. Projeto que cruza dados de vendas ganhas e perdidas, gera um Perfil de Cliente Ideal (ICP) com IA, realiza scoring inteligente de leads, identifica gargalos financeiros e prioriza oportunidades reais por vendedor e equipamento.
